# Visual Clock - 可视化时钟

基于 HTML5 Canvas + CSS3 开发的纯前端可视化时钟桌面应用，无需安装任何依赖，浏览器打开即可运行。具备模拟表盘、数字时钟、主题切换、番茄钟、多闹钟、秒表、世界时钟七大功能模块，**一屏展示，无需滚动**。

## 功能特性

| 功能 | 说明 |
|------|------|
| 模拟表盘 | Canvas 绘制渐变表盘、60格刻度、1~12数字标记、三根指针，秒针毫秒级平滑动画，**270px 大尺寸** |
| 数字时钟 | **40px** 等宽字体同步显示当前时间 HH:mm:ss |
| 日期显示 | **16px** 年月日 + 中文星期，与表盘同步刷新 |
| 主题切换 | **6 套精美主题**（经典简约 / 赛博朋克 / 暗黑奢华 / 马卡龙甜心 / 极光蓝调 / 复古怀旧），一键切换，表盘与界面同步变色 |
| 🌈 **动态浮动背景** | **5 个巨大彩色光晕球**（300~500px），鼠标移动时产生 **视差浮动效果**（移动幅度 35~60px），透明度 0.7，与 6 套主题颜色联动 |
| 🍅 **番茄钟** | **44px** 倒计时，25/5/15 分钟模式，自动循环，进度条，白噪音（雨声/咖啡馆/风声），**100px 全屏专注模式**隐藏其他 UI |
| 多闹钟列表 | 可添加多个闹钟，**5 种铃声可选**（双音风铃/经典电话/急促警报/学校铃声/音乐盒），**支持试听**，到点循环响铃 + 自定义弹窗，用户点击「知道了」关闭并移除 |
| 秒表计时 | **34px** 显示，开始/暂停/继续/重置/记圈，毫秒精度 |
| 世界时钟 | **多城市列表管理**（15个城市可选），支持增删，实时显示昼夜状态（白天☀️/夜晚🌙背景变化），**20px** 大号时间显示，时区差值显示 |

## 技术栈

- **语言**: HTML5 + CSS3 + JavaScript (ES6+)
- **图形**: Canvas 2D API
- **音效**: Web Audio API（白噪音生成 + 铃声合成）
- **样式**: CSS 变量 + 毛玻璃效果 + 渐变色 + CSS Grid 网格布局
- **动效**: CSS `backdrop-filter` 毛玻璃 + CSS 视差浮动 + `requestAnimationFrame` 逐帧渲染
- **渲染**: requestAnimationFrame（表盘）+ setInterval（数字时钟/番茄钟/闹钟）

## 操作指南

### 🎨 主题切换（右上角）
- **下拉菜单** 选择 6 套主题之一
- **彩色圆点** 直观点击切换
- **经典简约**主题额外支持浅色/深色一键切换
- 切换主题时，背景浮动光晕球的颜色也随之变化

### 🌈 动态浮动背景
- 页面背景有 **5 个半透明彩色光晕球**（300~500px 大小）
- **移动鼠标** → 光晕球朝反方向平滑浮动，产生 3D 景深感
- **离开窗口** → 光晕球自动复位
- 每套主题都有专属的光晕配色方案

### 🍅 番茄钟
1. **模式选择**：🎯 25分钟（专注） / ☕ 5分钟（短休息） / 🌿 15分钟（长休息）
2. **开始/暂停/重置**：点击控制，**44px** 大号倒计时
3. **自动循环**：专注→休息→专注...每4个专注后自动切换长休息
4. **进度条**：实时显示倒计时进度
5. **白噪音**：🌧 雨声 / ☕ 咖啡馆 / 🍃 风声，支持音量调节
6. **⛶ 全屏专注**：点击进入全屏模式，**100px** 巨幅倒计时 + 励志名言，按 ESC 退出

### ⏰ 多闹钟列表 + 铃声选择
1. **选择铃声**：🔔 双音风铃 / 📞 经典电话 / 🚨 急促警报 / 🔔 学校铃声 / 🎵 音乐盒
2. **▶ 试听**：点击预览当前选中的铃声效果
3. **设置时间** → 点击「➕ 添加闹钟」
4. **到点提醒流程**：
   - 🔊 **循环响铃**（每 1.8 秒重复一次）
   - 🖥️ **弹出精美模态框**（显示大号时间 + 毛玻璃动画背景）
   - 🔔 用户点击 **「知道了」** → 铃声停止 + 闹钟自动移除
5. **即将响铃**：30分钟内到点的闹钟显示「⏳ 即将响铃」标签
6. 点击 ✕ 可随时手动删除闹钟

### ⏱ 秒表计时
- **开始/暂停/继续** — **34px** 毫秒精度显示
- **记圈** — 记录每圈时间
- **重置** — 清零

### 🌍 世界时钟（多城市管理）
- **添加城市** — 从下拉列表选择15个城市点击添加
- **昼夜显示** — 根据当地时间自动切换白天/夜晚背景
- **时区差值** — 每个城市旁显示与本地时间的时差（如 +8.0h / -5.0h / ⏺ 同区）
- **大号时间** — **20px** 城市时间显示，清晰易读
- **删除城市** — 点击 ✕ 移除

## 核心实现说明

### 时钟绘制
- **表盘**: Canvas `createRadialGradient` 径向渐变填充，双圈装饰
- **刻度**: 60格，每5格为大刻度（粗线），其余小刻度（细线）
- **指针角度计算**:
  - 时针: `30° × (hour % 12) + 0.5° × minute`
  - 分针: `6° × minute + 0.1° × second`
  - 秒针: `6° × second + 6° × (ms/1000)` — 毫秒插值实现丝滑动画
- **刷新**: `requestAnimationFrame` 逐帧渲染，秒针流畅无卡顿

### 主题系统
通过 CSS 变量体系，6 套主题各自定义完整的颜色方案（背景、卡片、按钮、表盘、刻度、指针、昼夜、番茄钟状态、浮动光晕等），切换时动态替换 `body` 的 class。

### 动态浮动背景
- 5 个 `fixed` 定位的 div 元素，`radial-gradient` 径向渐变 + `blur(30px)` 高斯模糊
- 鼠标移动时计算视差偏移量：`offset = -direction × depth`，通过 `translate()` 驱动
- 使用 `cubic-bezier` 缓动函数实现丝滑跟随
- 光晕颜色随主题自动切换（CSS class 选择器）

### 番茄钟
- 三种模式通过定时器倒计时，支持暂停/继续
- 自动循环逻辑：专注→短休息×3→长休息→专注...
- 倒计时结束时播放提示音（Base64编码WAV）
- 白噪音通过 Web Audio API 生成不同类型的噪声，经低通滤波处理更自然

### 闹钟系统
- **5 种铃声** 通过 Web Audio API 合成（正弦波/方波/锯齿波组合）
- **AudioContext 保活**：创建无声循环缓冲区，保持音频上下文始终运行，绕过浏览器自动暂停策略
- **首次点击初始化**：用户首次点击页面时自动创建 AudioContext
- **到点流程**：`setInterval` 每秒检测 → 触发铃声 + 弹出模态框 → 用户点击「知道了」→ 停止铃声并移除闹钟

### 世界时钟
利用 `Date.now() + getTimezoneOffset()` 获取 UTC 时间戳，加上时区偏移秒数得到当地时间。根据小时数（6~18点）判断昼夜状态，计算与本地时区的差值。
