# Visual Clock - 可视化时钟

基于 HTML5 Canvas + CSS3 开发的纯前端可视化时钟桌面应用，无需安装任何依赖，浏览器打开即可运行。具备模拟表盘、数字时钟、主题切换、多闹钟、秒表、世界时钟六大功能模块。

## 功能特性

| 功能 | 说明 |
|------|------|
| 模拟表盘 | Canvas 绘制渐变表盘、60格刻度、1~12数字标记、三根指针，秒针毫秒级平滑动画 |
| 数字时钟 | 大号等宽字体同步显示当前时间 HH:mm:ss |
| 日期显示 | 年月日 + 星期，与表盘同步刷新 |
| 主题切换 | **6 套精美主题**（经典简约 / 赛博朋克 / 暗黑奢华 / 马卡龙甜心 / 极光蓝调 / 复古怀旧），一键切换，表盘与界面同步变色 |
| 多闹钟列表 | 可添加多个闹钟，列表管理，即将响铃提示，到点弹窗提醒，支持删除 |
| 秒表计时 | 开始/暂停/继续/重置/记圈，毫秒精度，圈数列表 |
| 世界时钟 | **多城市列表管理**（15个城市可选），支持增删，实时显示昼夜状态（白天☀️/夜晚🌙背景变化），时区差值显示 |

## 技术栈

- **语言**: HTML5 + CSS3 + JavaScript (ES6+)
- **图形**: Canvas 2D API
- **样式**: CSS 变量 + 毛玻璃效果 + 渐变色
- **渲染**: requestAnimationFrame（表盘）+ setInterval（数字时钟）

## 使用方法

**直接双击 `index.html` 在浏览器中打开即可运行！**

无需安装任何软件或依赖。

## 操作指南

### 🎨 主题切换（右上角）
- **下拉菜单** 选择 6 套主题之一
- **彩色圆点** 直观点击切换
- **经典简约**主题额外支持浅色/深色一键切换

### ⏰ 多闹钟列表
1. 选择时间 → 点击「➕ 添加闹钟」
2. 闹钟加入列表，30分钟内即将响铃的显示「⏳ 即将响铃」
3. 到点自动弹窗提醒，并从列表移除
4. 点击 ✕ 可随时删除闹钟

### ⏱ 秒表计时
- **开始/暂停/继续** — 毫秒精度计时
- **记圈** — 记录每圈时间
- **重置** — 清零

### 🌍 世界时钟（多城市管理）
- **添加城市** — 从下拉列表选择15个城市点击添加
- **昼夜显示** — 根据当地时间自动切换白天/夜晚背景
- **时区差值** — 每个城市旁显示与本地时间的时差（如 +8.0h / -5.0h / ⏺ 同区）
- **删除城市** — 点击 ✕ 移除

## 核心实现说明

### 时钟绘制
- **表盘**: Canvas `createRadialGradient` 径向渐变填充，双圈装饰
- **刻度**: 60格，每5格为大刻度（粗线），其余小刻度（细线）
- **指针角度计算**:
  - 时针: `30° × (hour % 12) + 0.5° × minute`
  - 分针: `6° × minute + 0.1° × second`
  - 秒针: `6° × second + 6° × (ms/1000)` — 毫秒插值实现丝滑动画
- **刷新**: `requestAnimationFrame` 逐帧渲染，秒针流畅无卡顿

### 主题系统
通过 CSS 变量体系，6 套主题各自定义完整的颜色方案（背景、卡片、按钮、表盘、刻度、指针、昼夜等），切换时动态替换 `body` 的 class。

### 世界时钟
利用 `Date.now() + getTimezoneOffset()` 获取 UTC 时间戳，加上时区偏移秒数得到当地时间。根据小时数（6~18点）判断昼夜状态，计算与本地时区的差值。
